# 前言
知彼知己者，百战不殆；不知彼而知己，一胜一负；不知彼，不知己，每战必殆。

# 环境
windows10（linux自测），python 3.8
# 用法
在modu目录下新建一个__init__.py的文件
运行e7_information.py即可
# 功能
1、域名反查ip
2、检测是否存在CDN
3、cms指纹识别
4、全端口扫描
5、子域名扫描
6、后台目录扫描
